===============================================================================================================================================
============================================================= Cara menjalankan Program ========================================================
===============================================================================================================================================

1. memasukan file ke htdoct lalu cari dengan nama folder yang ada di htdoct dan add database dengan nama db_thebaliestate
2. lalu cari saja di broser dan langsung di sambut dengan index.php
3. jika ingin melihat property tinggal pilih menu yang ada di atas 
4. jika tertarik , bisa hubguni agen lewat whatsapp
5. lalu ada menu property baru itu ada dikarenakan supaya terlihat property yang baru di rilis oleh agen
6. lalu ada menu agen itu untuk melihat agen kita
7. ada pilihan login itu untuk login ke admin page atau system dari websitenya, yang bisa login yaitu admin, agen, owner
8. ada menu register yaitu untuk membuat akun , yang perlu register hanya owner dan agen, pembeli tidak usah untuk register.
9. ada pilihan kontak kami , disana belum jalan karena masih beta
10. jika login dengan admin akan menuju ke admin page di sana admin hanya bisa menghapus mengedit man melihat data
11. jika login dengan agen akan melakukan validasi dengan property yang di tawarkan oleh owner dan melihat property yang dia pegang
12. jika login dengan owner / pemilik ia hanya bisa menginput data property dan menunggu property di sejutui oleh pihak agen.
